    class Methadone::Error

    Standard exception you can throw to exit with a given status code. Generally, you should prefer Methadone::ExitNow#exit_now! over using this directly, however you may wish to create a rich hierarchy of exceptions that extend from this in your app, so this is provided if you wish to do so.
